App uses https://holidayapi.com to get a list of holidays. Unfortunately, the free account doesn't allow get a list for the current year (only the previous one).
So for testing please use dates in 2021.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
